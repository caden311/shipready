# ShipReady

A free SEO and AEO (Answer Engine Optimization) scanner. Runs 35+ checks across 7 categories to verify a site is discoverable by both search engines and AI assistants.

## Live

[vientapps.com/tools/seo-check](https://vientapps.com/tools/seo-check/)

## Tech Stack

- Astro
- TypeScript
- Cloudflare Workers
- Regex-based HTML parsing

## Categories Checked

Meta and head, discoverability, on-page structure, social sharing, performance, structured data, and AEO (LLM-readable signals).
